Summarize the project and what problem it was solving.
This project was took an input file and then saved the frequencies of each unique word in a map and an additional file. Then there was a menu 
that allowed you to: choose to search for a specific word, print the entire list to a console, or print the entire list as a histogram to the console.

I felt like I did the menu quite well and also the creation of the map through iterating the file.
